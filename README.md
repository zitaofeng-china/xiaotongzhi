# 备忘录应用

一个基于 React 18 + JavaScript + Vite 构建的现代化备忘录前端应用。

## 功能特性

- ✅ 添加备忘录（支持标题和内容）
- ✏️ 编辑备忘录
- 🗑️ 删除备忘录
- ☑️ 标记完成/未完成
- 💾 本地存储（数据自动保存到浏览器）
- 📱 响应式设计（支持移动端和桌面端）
- 🎨 现代化UI设计

## 技术栈

- React 18
- JavaScript
- Vite
- CSS3

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

应用将在 `http://localhost:5173` 启动

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

## 使用说明

1. **添加备忘录**：在表单中输入标题（可选）和内容，点击"添加备忘录"按钮
2. **编辑备忘录**：点击备忘录卡片或编辑按钮，修改内容后点击"保存修改"
3. **标记完成**：点击复选框来标记备忘录为完成状态
4. **删除备忘录**：点击删除按钮删除不需要的备忘录

所有数据会自动保存到浏览器的本地存储中，刷新页面后数据不会丢失。
